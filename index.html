Advanced HTML5 Placeholder Polyfill

Provides <input placeholder="..."> support for IE 9 (and some prior versions)

View on GitHub Download .zip Download .tar.gz

Getting Started

Simply embed the following code in your page, after you've included jQuery, and preferably at the bottom of the page:

  <!--[if lte IE 9]><script src="//yumpdigital.github.io/Advanced-HTML5-Placeholder-Polyfill/js/jquery.placeholder.js"></script><![endif]-->
